Fatos sobre o Stomatopoda

Informações gerais

O Stomatopod é um animal muito intressante porque ele é pequeno mas é capaz de fazer muitas coisas, seu nome científico é Odontodactylus scyllarus. Ele é caracterizado por ser extremamente agressivo, presente há 400 milhões de anos atrás.
